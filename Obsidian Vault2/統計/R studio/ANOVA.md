
### 第一部分：ANOVA 的本質（訊號雜訊比）

不要去想複雜的公式，ANOVA 的核心邏輯只有一句話：

**「我們觀察到的數據差異（Total Variance），有多少是因為我們的實驗操弄（Effect/Signal），又有多少只是隨機誤差（Error/Noise）？」**

想像你在做 fMRI 實驗：

1. **SST (Total Sum of Squares - 總變異):** 數據裡的總波動。
    
2. **SSB (Between-group - 組間變異):** 「實驗組」跟「對照組」的差別。這是你要的 **Signal**。
    
3. **SSW (Within-group - 組內變異):** 同一組受試者內部每個人的差異。這是你無法控制的 **Noise**。
    

**F 值 (F-ratio) 就是訊號雜訊比：**

$$F = \frac{\text{Signal (組間差異)}}{\text{Noise (組內差異)}}$$

- 如果 F 很大，代表實驗操弄的效果蓋過了隨機雜訊 -> **顯著**。
    
- 如果 F 很小，代表你的操弄效果被淹沒在個體差異的雜訊裡了 -> **不顯著**。

### 第二部分：為什麼會有 Type I, II, III？（蛋糕怎麼切？）

這個問題只有在**「多因子 ANOVA（比如 2-way ANOVA）」**且**「樣本數不平衡（Unbalanced Data）」**時才會出現。

如果你的實驗設計是完美的（例如：男生 10 人、女生 10 人，且每組吃藥/不吃藥的人數都一樣），那 Type I, II, III 算出來的結果完全一樣。

**但在現實世界（你的 PS150 研究）中**，如果有受試者退出，導致 A 組 10 人、B 組 8 人，這時兩個因子（例如：性別、藥物）之間會產生**重疊的解釋力（Collinearity）**。

這就像一塊重疊的蛋糕（變異量），到底要算給誰？

#### 1. Type I SS (Sequential - 序列式)：「先搶先贏」

- **邏輯：** 依照你放入模型的順序切蛋糕。
    
- **情況：** 模型是 `Y ~ A * B`。
    
    - 因子 A 先進場，把屬於它的（包含跟 B 重疊的部分）全部拿走。
        
    - 因子 B 後進場，只能拿剩下的。
        
    - 最後剩的給交互作用。
        
- **缺點：** 把 `A` 放前面跟把 `B` 放前面，結果會不一樣！這在科學上是荒謬的，所以**除了特定巢狀設計（Nested Design）外，現代學術界極少使用**。
    
- **誰預設用它？** R 的 `aov()` 和 `lm()`。這就是為什麼用 Base R 跑不平衡資料會出錯。
    

#### 2. Type II SS (Hierarchical - 階層式)：「有條件的公平，忽略交互作用」

這就是你問的重點。

- **邏輯：**
    
    - 計算 A 的貢獻時，會控制 B 的影響（扣除 B 獨有的部分）。
        
    - 計算 B 的貢獻時，會控制 A 的影響（扣除 A 獨有的部分）。
        
    - **關鍵點：** 它在計算主效應（Main Effects）時，**假設交互作用（Interaction）不存在（或忽略交互作用的重疊部分）**。
        
- **優點：** 如果你**非常確定**兩個因子之間沒有交互作用，Type II 的檢定力（Power）其實是最高的。它專注於檢驗主效應。
    
- **什麼時候用？** 在某些探索性研究，或者你只想看主效應強不強，不在乎交互作用時。
    
- **缺點：** 如果真的有交互作用，Type II 對主效應的估計可能會失準。
    

#### 3. Type III SS (Marginal - 邊際式)：「極度保守，SPSS 的最愛」

- **邏輯：**
    
    - 計算 A 的貢獻時，控制 B **以及** 交互作用 (A:B)。
        
    - 計算 B 的貢獻時，控制 A **以及** 交互作用 (A:B)。
        
    - 大家都不准拿「重疊」的部分，每個人只拿「完全屬於自己獨有」的那一塊變異量。
        
- **優點：** 最保守，不管你順序怎麼放，結果都一樣。它回答的是科學家最在乎的問題：「在控制了其他所有因素（包含交互作用）後，這個因子是否依然顯著？」
    
- **缺點：** 因為把重疊部分丟掉，如果樣本數極度不平衡，檢定力會稍微下降。但為了正確性，這是必須的犧牲。
    
- **誰預設用它？** SPSS、SAS、Stata。

### 圖解比喻：聚光燈下的舞台

想像舞台上有兩個演員 A 和 B，以及一盞聚光燈（總變異量）。

因為樣本不平衡，A 和 B 站得有點近，影子（解釋力）重疊了。

- **Type I (偏心導演):** 導演喊「A 先來！」，A 的影子連同重疊部分都算 A 的。B 只能算剩下沒重疊的影子。
    
- **Type II (和平主義者):** 導演說「你們只比誰的主體影子大，重疊的部分先不管，假設你們沒有互動（交互作用）。」
    
- **Type III (嚴格裁判 - SPSS):** 導演說「重疊的通通不算！A 只能算完全乾淨的 A 影子，B 只能算完全乾淨的 B 影子。」

### 1. 既然 Type III 最保險，那 Type I 和 Type II 到底什麼時候用？

雖然 Type III 是社科與生醫領域的「預設值」，但 Type I 和 Type II 並不是廢物，它們在特定的科學邏輯下非常有用。

#### **Type I SS (序列式)：用於「階層迴歸」或「多項式分析」**

當變數進入模型的**順序具有強烈的理論意義**時，必須用 Type I。

- **情境 A：多項式回歸 (Polynomial Regression)**
    
    你想看時間對腦波的影響是線性的，還是曲線的？
    
    - 模型：`y ~ x + x^2 + x^3`
        
    - 邏輯：你要先確定「一次方（線性）」解釋了多少，剩下的殘差才輪得到「二次方（曲線）」去解釋，最後才是「三次方」。如果你先算三次方，物理意義就亂了。這裡**順序不能動**，所以用 Type I。
        
- **情境 B：控制變項的階層分析 (Hierarchical Regression)**
    
    這在心理學很常見。
    
    - 假設你想證明你的「新療法 (B)」有效。
        
    - Reviewer 要求：「請證明你的療法在控制了『年齡 (A)』之後依然有效。」
        
    - 分析策略：先讓「年齡」把變異量吃光（不管它跟療法有沒有重疊，年齡優先），剩下的殘渣如果你的「新療法」還能吃得到且顯著，那才是真的厲害。
        
    - 這時候你會刻意把 `Age` 放前面，`Treatment` 放後面，並使用 Type I。
        

#### **Type II SS (階層式)：用於「模型篩選」或「檢定力最佳化」**

Type II 的假設是**「沒有交互作用」**。

- **情境：探索性研究或交互作用不顯著時**
    
    如果你跑了 Type III，發現交互作用項的 p 值是 0.85（超級不顯著）。
    
    這時候，交互作用項佔用的「自由度」和「變異量計算權重」其實是在浪費你的統計檢定力（Power）。
    
- **用法：** 很多統計學家建議，一旦確定交互作用不顯著，應該把交互作用項拿掉，或者改用 Type II 來檢定主效應（Main Effects）。因為在沒有交互作用的前提下，Type II 對主效應的檢定力比 Type III 更強（更容易發現顯著差異）。

### 2. 什麼是「球形檢定 (Mauchly's Test of Sphericity)」？

這是**重複測量 ANOVA (Repeated Measures ANOVA)** 專屬的「門神」。如果你不懂它，跑出来的 p 值可能是虛假的。

#### **核心概念：變異數的「公平性」**

想像你做了一個實驗，測量三個時間點：前測 (T1)、中測 (T2)、後測 (T3)。

- **獨立樣本 ANOVA 的要求：** 變異數同質性（各組的變異程度要差不多）。
    
- **重複測量 ANOVA 的要求：** **球形假設 (Sphericity)**。
    
    它不是比各組的變異數，而是比**「時間點之間的差異分數」的變異數**。
    

具體來說，球形假設要求：

$$Var(T1 - T2) \approx Var(T1 - T3) \approx Var(T2 - T3)$$

也就是說，「前測跟中測的變化幅度」、「前測跟後測的變化幅度」……這些變化幅度的波動程度應該要差不多。

#### **為什麼這很重要？**

如果不符合球形假設（例如：T1 跟 T2 很像，差異很小；但 T3 突然波動劇烈），原本 ANOVA 依賴的 F 分配就會失準。

- **後果：** 你的 **Type I Error（型一錯誤）會暴增**。
    
- **白話文：** 電腦算出來 p = 0.03 (顯著)，但實際上可能只有 p = 0.08 (不顯著)。你會高興地發表結果，然後發了一篇偽科學論文。
    

#### **怎麼解決？（SPSS 與 R 的做法）**

當我們跑重複測量 ANOVA 時，第一步先看 **Mauchly's Test**。

1. **若 p > .05 (不顯著)：** 恭喜！符合球形假設。直接看標準的 "Sphericity Assumed" 那一行的結果。
    
2. **若 p < .05 (顯著)：** 慘了，違反球形假設。
    
    這時候不能看原本的 p 值，必須進行**「校正 (Correction)」**。
    

#### **校正方法：對 p 值進行「懲罰」**

既然原本的 p 值太容易顯著（太寬鬆），我們就要讓它變嚴格一點。我們通過調整自由度（df）來實現，這就是你會看到的：

- **Greenhouse-Geisser (GG) 校正：** 最常用，比較保守（懲罰比較重）。
    
- **Huynh-Feldt (HF) 校正：** 比較寬鬆（懲罰比較輕）。
    

**實戰判斷法則：**

- 如果違反球形假設，通常直接看 **Greenhouse-Geisser** 校正後的 p 值。如果校正後還顯著，那就是真的顯著。


#### 🌟 完整邏輯斯諦迴歸公式

將 $Z$ 的線性預測式代入上式，我們得到了邏輯斯諦迴歸的完整表達式：

$$\text{機率 } P(Y=1 | \mathbf{X}) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 X_1 + \beta_2 X_2 + \dots + \beta_k X_k)}}$$

這個公式就像線性迴歸的 $Y = \beta_0 + \beta_1 X_1 + \dots$ 一樣，代表了邏輯斯諦迴歸的**完整模型**。它直接說明了：

1. **左側：** 預測變量 $\mathbf{X}$ 條件下，結果 $Y=1$ 的機率。
    
2. **右側：** 如何通過 S 型曲線將線性得分 (指數的負號後面部分) 轉換為機率。
    

這個式子就是您尋找的、能代表邏輯斯諦迴歸的單一式子。

### 🌟 一、基礎定義與目的

|**項目**|**說明**|
|---|---|
|**定義**|屬於**廣義線性模型 (Generalized Linear Model, GLM)** 的一種，用於估計事件發生的**機率 (Probability)**。|
|**結果變量**|必須是**類別**數據，最常見的是**二元 (Binary)**，例如：是/否、成功/失敗、得病/未得病 (通常編碼為 1/0)。|
|**輸入變量**|可以是連續、類別或兩者的組合。|
|**核心目的**|根據一組預測變量，預測特定結果事件發生的機率 $P(Y=1)$，並量化這些預測變量的**影響強度**。|

### 📈 二、模型結構與核心函數

標準線性迴歸預測的是連續值 $Y$，而邏輯斯諦迴歸預測的是機率 $P$ (範圍在 0 到 1 之間)。

#### 1. 線性預測式 (Linear Predictor)

這是模型的「核心得分」，與標準線性迴歸的右側相同：

$$Z = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \dots + \beta_k X_k$$
$Z$ 本身是一個連續的、無上下限的數值 (範圍從 $-\infty$ 到 $+\infty$)。誤區釐清$Z$ 不是結果變量 $Y$ 或其族群數量。在邏輯斯諦迴歸中， $Z$ 是一種潛在得分或潛在變量，代表事件發生的對數勝算 (Log-Odds)。
#### 2. 邏輯函數 (Logit Function)

由於機率 $P$ 必須介於 0 和 1 之間，我們不能直接用 $Z$ 來預測 $P$。邏輯斯諦迴歸透過以下兩個步驟實現：

|**步驟**|**數學表達式**|**統計意義 (Link Function)**|
|---|---|---|
|**勝算 (Odds)**|$\text{Odds} = \frac{P}{1-P}$|將機率 (0 到 1) 轉換為**勝算** (0 到 $\infty$)。|
|**對數勝算 (Log-Odds)**|$\ln\left(\frac{P}{1-P}\right) = Z$|這是**連結函數 (Link Function)**，將非線性的機率與線性的預測式 $Z$ 連結起來。|
|**機率 (Probability)**|$P = \frac{1}{1 + e^{-Z}}$|這是**S型曲線 (Sigmoid Function)**，將 $Z$ (負 $\infty$ 到正 $\infty$) 轉換回機率 $P$ (0 到 1)。|
為了將無限範圍的 $Z$ 轉換到 $[0, 1]$ 區間，統計學家採取了「勝算 (Odds)」作為中間步驟：

|**步驟**|**說明**|**數學表達式**|**範圍**|
|---|---|---|---|
|**A. 定義機率 ($P$)**|事件發生的機率。|$P$|$[0, 1]$|
|**B. 定義勝算 (Odds)**|事件發生與不發生機率的比值。|$\text{Odds} = \frac{P}{1-P}$|$[0, \infty]$|
|**C. 定義對數勝算 (Log-Odds)**|對勝算取自然對數 ($\ln$)。|$\ln\left(\frac{P}{1-P}\right)$|$[-\infty, +\infty]$|

**邏輯函數 (Logit Function)** $\ln\left(\frac{P}{1-P}\right)$ 就是**對數勝算**。由於它的範圍是 $(-\infty, +\infty)$，因此它可以被**線性預測式 $Z$ 所代表**：

$$Z = \ln\left(\frac{P}{1-P}\right)$$

這就是**邏輯函數 (Logit Function)** 的來源和目的，它是邏輯斯諦迴歸的**連結函數 (Link Function)**。
### 🔑 三、係數解釋與結果呈現

邏輯斯諦迴歸的係數 $\beta_i$ 並**不是**直接解釋 $Y$ 改變多少，而是解釋**對數勝算 (Log-Odds)** 的改變。

|**項目**|**數學表達式**|**實質解釋 (最重要的輸出)**|
|---|---|---|
|**迴歸係數 ($\beta_i$)**|$\beta_i$|$X_i$ 每增加一個單位，**對數勝算**改變 $\beta_i$ 個單位。|
|**勝算比 (Odds Ratio, OR)**|$\text{OR}_i = e^{\beta_i}$|$X_i$ 每增加一個單位，**事件發生的勝算**將**乘以** $\text{OR}_i$。|
|**OR 解釋**|||
|**$\text{OR} > 1$**|$e^{\beta_i} > 1$|$X_i$ 增加，事件發生的勝算**增加** (正相關)。|
|**$\text{OR} < 1$**|$e^{\beta_i} < 1$|$X_i$ 增加，事件發生的勝算**減少** (負相關)。|
|**$\text{OR} = 1$**|$e^{\beta_i} = 1$|$X_i$ 增加，對事件發生的勝算**沒有影響**。|
邏輯斯諦迴歸之所以產出 **OR**，是**基於其數學原理（Logit 連結函數）的設計結果**。它在數學上優雅地將無限範圍的線性預測式與有限範圍的機率連結起來。
### 🛠️ 四、模型評估與診斷

| **評估項目**                     | **說明**                                                  | **目的**                                      |
| ---------------------------- | ------------------------------------------------------- | ------------------------------------------- |
| **Wald 檢定 (Wald Test)**      | 用於檢定單一係數 $\beta_i$ 是否顯著不為零 (即 $\text{OR}$ 是否顯著不為 1)。    | 判斷單一預測變量的貢獻是否顯著。                            |
| **擬合優度檢定 (Goodness-of-Fit)** | 常見的有**Hosmer-Lemeshow 檢定**。                             | 檢定模型是否與數據擬合良好。                              |
| **ROC 曲線**                   | 繪製**真陽性率 (Sensitivity)** 與**假陽性率 (1-Specificity)** 的曲線。 | 評估模型區分 0 和 1 兩組的能力 (判斷閾值調整)。                |
| **AUC**                      | **曲線下面積 (Area Under the Curve)**。                       | 數值範圍 0.5 (隨機猜測) 到 1.0 (完美預測)。用於量化模型的整體預測效能。 |
| **最大概似法**                    | **Maximum Likelihood Estimation (MLE)**                 | 與線性迴歸使用最小平方不同，邏輯斯諦迴歸使用 MLE 來估計係數。           |

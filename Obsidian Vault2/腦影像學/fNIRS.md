# functional near-infrared spectroscopy
大部分只能測pre frontal的位置

功能性近紅外光譜主要反映感興趣區 (ROI) 內血紅蛋白濃度的變化，因此幾乎所有納入的研究中，氧合血紅蛋白濃度變化 (△oxyHb) 都是最常研究的參數，也有研究分析了 ROI 的功能性 (FC)（Bu [et al., 2019](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B7) [;](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B9) Canario [et](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B16) [al.,](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B27) 2022 ; [Nguyen et al.,](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B34) 2019 ; [Yang and Hong, 2021](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B54) ; [Yu et al., 2020](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B63) ; [Zhang et al., 2022](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B66)），△oxyHb 的斜率也是一個重要參數（[Ung et al., 2020](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B50) ; [Li R. et al., 2018](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B26)）。 Niu [et al. (2019)](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B36)選擇動態 FC 變異性作為研究參數，其定義為低頻（<0.1 Hz）範圍內功率譜的曲線下面積。此外，一項研究分析了 △oxyHb 的多尺度熵 (MSE)（[Li X. et al., 2018](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B28)）。一些研究也觀察到了側化現象，即腦區活化的不對稱性（[Haberstumpf et al., 2022](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B17)；[Yeung et al., 2016b](https://pmc.ncbi.nlm.nih.gov/articles/PMC12014625/#B59)）。

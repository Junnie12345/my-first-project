4. 混合模型 (Mixed Models / HLM / LMM)

4.1 核心定義

混合模型包含 **固定效應 (Fixed Effects)** 與 **隨機效應 (Random Effects)**。

• **線性混合模型 (LMM):** 當反應變數是連續且服從常態分佈時（GLM 的常態分佈延伸）。

• **廣義線性混合模型 (GLMM):** 當反應變數是非常態（如二元、計數）時（GLM 的延伸）。

4.2 參數意義

• **固定效應 (Fixed Effects):** 描述整個母體的平均趨勢（例如：藥物的平均療效）。這相當於傳統迴歸的 β。

• **隨機效應 (Random Effects,** bi​**):** 描述特定群集（人、學校）偏離母體平均的程度。

    ◦ _Random Intercept:_ 每個人的起點不同（例如：每個人的基礎血壓不同）。

    ◦ _Random Slope:_ 每個人對變項的反應不同（例如：每個人對藥物的反應速率不同）。

4.3 兩階段建模 (Two-Stage Modeling)

Laird & Ware (1982) 將其描述為：

1. **第一階段（個體）：** 建立個人的迴歸方程式 Yi​=Xi​α+Zi​bi​+ei​。

2. **第二階段（母體）：** 假設個體參數 bi​ 來自一個常態分佈 bi​∼N(0,D)。

4.4 關鍵特性：收縮 (Shrinkage) 與部分池化 (Partial Pooling)

混合模型會利用母體分佈的資訊來「借力」給樣本數少的個體。如果某個群集的數據很少（如某間學校只有3個學生），模型會將其估計值向**總體平均數**拉近（Shrinkage），避免過度解讀極端值。這使得混合模型在處理**非平衡資料 (Unbalanced Data)** 時非常強大。

4.5 估計方法

• **ML (Maximum Likelihood):** 用於估計，但會低估變異數。

• **REML (Restricted ML):** 混合模型的標準估計法，能提供無偏的變異數估計。

5. GEE 與 混合模型 (Mix Models) 的詳細比較

|   |   |   |
|---|---|---|
|特性|**GEE (廣義估計方程式)**|**Mixed Models (混合模型 / LMM)**|
|**別名**|邊際模型 (Marginal), 母體平均 (PA)|條件模型 (Conditional), 特定主體 (SS), HLM|
|**推論目標**|**群體平均值**：藥物平均對大眾有效嗎？|**特定個體預測**：藥物對_這位_病人有效嗎？|
|**參數解釋 (**β**)**|βmarginal​ (通常數值較小)|βconditional​ (通常數值較大)|
|**相關性處理**|視為干擾 (Nuisance)，用工作矩陣處理|視為感興趣的變異，用隨機效應建模|
|**分佈假設**|弱假設 (Semi-parametric)，不需指定完整分佈|強假設 (Parametric)，需假設隨機效應服從常態分佈|
|**缺失值處理**|傳統需 MCAR，但結合多重插補 (MI) 後表現優異|假設 MAR，內建處理能力強，但對插補較敏感|
|**計算與收斂**|計算簡單，收斂容易|計算複雜，可能遇到收斂問題 (Convergence issues)|
|**適用資料**|適合流行病學、政策評估|適合生長曲線、個體差異分析、遺傳育種 (GBLUP)|

5.1 "To GEE or Not to GEE" 的爭論

Hubbard 等人指出，混合模型依賴於無法驗證的假設（隨機效應必須是常態分佈）。如果這個假設錯了，結果就會偏誤。相比之下，GEE 是一種對真實狀況更安全的近似（Approximation of truth）。
## 📝 線性混合模型 (Linear Mixed Models, LMM) 核心重點筆記

### 🌟 一、LMM 定義與基礎

|**項目**|**說明**|
|---|---|
|**定義**|屬於**迴歸模型**的一種，用於分析資料中同時包含**固定效應**和**隨機效應**的結構。|
|**適用情境**|處理**重複測量**或**巢狀 (Nested)** 結構的數據，這些數據的觀察值之間存在相關性，違反了標準迴歸模型中「觀察值獨立」的假設。|
|**關鍵優勢**|能夠在一個模型中**同時考慮**群體平均趨勢 (固定效應) 和個體間差異/相關性 (隨機效應)。|
|**核心目的**|準確估計效應、控制共變數的影響，並**正確建模資料的相關性結構**。|

### 🔍 二、LMM 的兩大組成要素

LMM 將資料的變異分解為兩部分：

|**效應類型**|**說明 (NCBI 連結強調的重點)**|**範例**|
|---|---|---|
|**1. 固定效應**|**群體參數**，描述**所有受試者/單元共有的平均趨勢**。它們是研究者感興趣的**主要預測變量**或組別差異，其估計值是固定的。|藥物劑量、性別、處理組別、時間點等。|
|**2. 隨機效應**|描述由**未被固定效應解釋的相關性和變異性**。它們是從一個**更大的潛在群體**中隨機抽取的，用於捕捉**個體間的差異 (Intercept)** 或**個體對處理的反應差異 (Slope)**。|個體受試者 (Subject ID)、不同學校、不同家庭等。|
**X**跟**Z**分別為獨立變項矩陣
β代表固定效果(Fixed-effects)的常數向量
γ代表隨機效果(Random effects)的隨機向量
ε為誤差項

### 📊 三、LMM 的主要應用結構 (數據類型)

LMM 是分析相關性數據的強大工具，主要應用於以下兩種數據結構：

|**數據結構**|**說明**|**LMM 如何處理**|
|---|---|---|
|**重複測量 (Repeated Measures)**|同一個受試者在**不同時間點**或**不同條件下**被多次測量。數據是**縱向**的。|隨機效應（通常是**受試者 ID**）會捕捉同一受試者不同測量之間的**時間相關性**。|
|**巢狀數據 (Clustered/Nested Data)**|較小的單元被嵌套在較大的單元內。例如：學生在班級內，班級在學校內。|隨機效應（通常是**巢狀單位 ID**，如**班級或學校**）會捕捉同一個群集內觀測值之間的**群體相關性**。|

### 🛠️ 四、LMM 與其他統計模型的比較

|**模型**|**特點**|**不足之處 (LMM 的優勢)**|
|---|---|---|
|**標準線性迴歸 (OLS)**|僅包含**固定效應**。|假設觀察值**完全獨立**，不適用於相關數據。若用於相關數據會導致標準誤被低估，P 值不準確。|
|**重複測量 ANOVA**|傳統上處理重複測量。|需要數據完整、變異數/共變數結構的假設嚴格 (如球形條件)，且**無法處理不規則的、缺失的或不平衡的數據**。|
|**廣義估計方程 (GEE)**|處理相關數據，但只關注**群體平均效應**。|**不提供個體水平的差異** (隨機效應) 資訊。|

## 📐 五、線性混合模型 (LMM) 的數學模型與結構

線性混合模型 (LMM) 的精髓在於其將傳統線性迴歸的結構一分為二，分別用於**固定效應**和**隨機效應**。

### 1. LMM 的一般數學表達式

線性混合模型通常表示為：

$$\mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \mathbf{Z}\mathbf{u} + \boldsymbol{\epsilon}$$

這個公式是 LMM 的核心，它將觀察值 $\mathbf{y}$ 的變異來源清楚地劃分：

|**符號 (數學元素)**|**意義 (統計學解釋)**|
|---|---|
|$\mathbf{y}$|**因變量 (Dependent Variable)** 的向量。|
|$\mathbf{X}$|**固定效應**的設計矩陣 (Design Matrix)，包含自變量的值。|
|$\boldsymbol{\beta}$|**固定效應參數**的向量 (Fixed Effects)。這就是傳統迴歸模型的**迴歸係數**。|
|$\mathbf{Z}$|**隨機效應**的設計矩陣，定義了哪些隨機效應影響了哪些觀察值。|
|$\mathbf{u}$|**隨機效應參數**的向量 (Random Effects)。其平均值假設為 $\mathbf{0}$。|
|$\boldsymbol{\epsilon}$|**殘差項 (Residual Errors)** 的向量。捕捉所有模型未解釋的個體內變異。|

### 2. 模型核心分解 (與永析網頁概念結合)

正如永析統計網站提到 LMM 建立在基礎迴歸分析上，我們可以將 LMM 拆解成兩大部分：

#### **(A) 固定效應部分 ($\mathbf{X}\boldsymbol{\beta}$)**

- **功能：** 捕捉**群體平均趨勢**。
    
- **概念：** 類似於標準的線性迴歸 ($y = \beta_0 + \beta_1 x_1 + \dots$)，描述所有觀測單元 (例如所有受試者) 的**共同平均反應**。
    

#### **(B) 隨機效應部分 ($\mathbf{Z}\mathbf{u}$)**

- **功能：** 捕捉**個體差異**和**相關性**。
    
- **概念：** 這是 LMM 區別於標準迴歸的關鍵。它允許模型中的某些參數 (如截距或斜率) **在不同隨機單位之間變化**，從而處理重複測量或巢狀資料中的相關性。
    

### 3. 隨機項的假設 (變異數結構)

為了正確地處理相關性，LMM 對隨機項的分佈和結構作出了嚴格的假設：

|**項**|**假設分佈**|**意義**|
|---|---|---|
|**隨機效應 ($\mathbf{u}$)**|$\mathbf{u} \sim N(\mathbf{0}, \mathbf{G})$|隨機效應通常假設服從**平均值為零**的常態分佈。 $\mathbf{G}$ 是**隨機效應的共變異數矩陣**，捕捉了不同隨機效應之間的變異和相關性。|
|**殘差項 ($\boldsymbol{\epsilon}$)**|$\boldsymbol{\epsilon} \sim N(\mathbf{0}, \mathbf{R})$|殘差項也假設服從常態分佈。 $\mathbf{R}$ 是**殘差 (個體內) 的共變異數矩陣**，捕捉了同一隨機單位 (如同一受試者) 內不同時間點的**相關性**。|

> **💡 關鍵：** $\mathbf{G}$ (隨機效應變異) 和 $\mathbf{R}$ (殘差變異) 共同構成了 LMM 處理相關性和異質性的能力。研究者通常需要指定或評估這些**變異數-共變異數結構**。


## 🔬 六、MLMM 的深入解釋：多元線性混合模型 (Multivariate Linear Mixed Model)

當 MLMM (Multivariate Linear Mixed Model) 用於與 LMM 相同的重複測量或巢狀數據時，它提供了一個重要的延伸：**同時分析多個結果變量 (Multiple Outcomes)**。

### 1. 核心概念與目的

|**項目**|**說明**|
|---|---|
|**LMM (單變量)**|分析**單一**因變量 (Outcome) 的平均趨勢與變異性。|
|**MLMM (多變量)**|**同時分析兩個或多個**因變量 (例如：治療對**血壓**和**膽固醇**的影響)。|
|**關鍵優勢**|能夠估計並考量**不同結果變量之間**的**相關性**。這比分別運行多個單一 LMM 更有效率且更完整。|
|**研究目的**|檢定某個自變量（固定效應）對多個結果變量是否具有**整體影響 (Global Test)**，以及這些影響之間的協同作用。|

### 2. 數學模型的延伸

MLMM 的數學表達式與 LMM 類似，但其向量和矩陣的維度擴展到了 $p$ 個結果變量。

單變量 LMM：

$$\mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \mathbf{Z}\mathbf{u} + \boldsymbol{\epsilon}$$

(y 是一個包含所有單一結果測量的向量)

**多元 MLMM：**

其結構仍然是 $\mathbf{y}^{*} = \mathbf{X}^{*}\boldsymbol{\beta}^{*} + \mathbf{Z}^{*}\mathbf{u}^{*} + \boldsymbol{\epsilon}^{*}$，但矩陣和向量的維度包含了所有結果變量。

- **關鍵在於：**
    
    - **固定效應 ($\boldsymbol{\beta}^{*}$):** 每個結果變量都有自己的一組固定效應參數。
        
    - **隨機效應變異 ($\mathbf{G}^{*}$):** 隨機效應不僅捕捉**個體間變異**，還捕捉**個體間不同結果變量之間的相關性** (例如：一個人的高血壓截距和高膽固醇截距之間可能相關)。
        
    - **殘差變異 ($\mathbf{R}^{*}$):** 殘差項捕捉**個體內變異**，同時捕捉**同一時間點不同結果變量之間的殘差相關性**。
        

> **💡 核心差異：** MLMM 不僅考慮了重複測量數據的**時間相關性**和**個體差異**，還額外考慮了**不同結果變量 (Outcomes) 之間的相互關聯**。

## 🏡 附註：MLMM 的另一種解釋：多水平模型 (Multilevel Model)

如果 MLMM (或更常用的 MLM) 指的是 **多水平線性模型 (Multilevel Linear Model)**，那麼它與 LMM 的關係是：

|**項目**|**說明**|
|---|---|
|**別稱**|**分層線性模型 (Hierarchical Linear Model, HLM)**。|
|**關係**|多水平模型 (MLM) 實際上是**線性混合模型 (LMM) 的一個子集或同義詞**，特別強調**巢狀結構**的應用。|
|**數據結構**|學生 (Level 1) 巢狀於 班級 (Level 2)，班級巢狀於 學校 (Level 3)。|
|**主要功能**|分析**不同層次 (Level)** 上的變異。例如，學生考試成績的變異可以分解為：來自**學生個體**的變異 (Level 1) + 來自**班級平均差異**的變異 (Level 2)。|

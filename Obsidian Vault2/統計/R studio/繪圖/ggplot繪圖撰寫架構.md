## 🛠️ 第一部分：圖層架構與撰寫順序

`ggplot2` 遵循「圖層疊加 (The Grammar of Graphics)」邏輯，建議依照以下順序撰寫，這能幫助你邏輯清晰地除錯：

1. **初始化 (Base Layer)**：定義資料源與全域對應 (`aes`)。
    
2. **輔助背景 (Background Layers)**：如 `geom_hline`，放在數據之下作為參考。
    
3. **主要數據幾何 (Data Layers)**：`geom_col` 或 `geom_point`，以及 `geom_errorbar`。
    
4. **比例尺與顏色 (Scales)**：調整 X/Y 軸範圍、標籤名稱、顏色映射。
    
5. **標籤說明 (Labels)**：標題、副標題、座標軸名稱。
    
6. **視覺風格 (Theme)**：整體配色主題、字體大小、邊距、格線。

## 🎨 第二部分：幾何圖層 (Geometries) 的細節

### 1. 柱狀圖 vs. 點區間圖

|**幾何物件**|**核心功能**|**常用參數說明**|
|---|---|---|
|**`geom_col()`**|畫出數值柱狀|`width`: 寬度 (0.9 為預設, 0.5 較修長)；`fill`: 填滿色；`alpha`: 透明度。|
|**`geom_point()`**|畫出平均值中心點|`size`: 點大小；`shape`: 形狀 (16 是實心, 21 是可換邊框色的空心)；`stroke`: 邊框粗細。|
|**`geom_errorbar()`**|畫出誤差線 (SD/SE)|**必須在 `aes()` 內定義 `ymin` 與 `ymax`**；`width`: 橫槓寬度。|

### 2. 輔助線圖層

- **`geom_hline(yintercept = ...)`**：水平線。
    
- **`geom_vline(xintercept = ...)`**：垂直線（在離散 X 軸中，數字代表第幾個類別，例如 1.5 代表 A 與 B 之間）。

## 📏 第三部分：量尺 (Scales) 與空間計算

這部分決定了圖表的「呼吸感」與「精準度」。

### 1. X 軸離散量尺 (`scale_x_discrete`)

- **`expand = expansion(add = 0.75)`**：
    
    - **邏輯**：X 軸每個組別的中點座標是整數 (1, 2, 3...)。
        
    - **計算**：`add = 0.75` 代表從中心點往左右各延伸 0.75 單位。若柱寬為 0.5，邊界淨空則為 $0.75 - 0.25 = 0.5$。
        
- **`labels`**：用 `c("原本名稱" = "展示名稱")` 進行重新命名。
    

### 2. Y 軸連續量尺 (`scale_y_continuous`)

- **`limits`**：強制設定數值範圍。**注意：超出此範圍的數據會被整圖移除。**
    
- **`breaks`**：設定刻度線出現在哪裡。
    
- **`expand = expansion(mult = c(0, 0.1))`**：下方不留白 (貼地)，上方留 10% 空間。
    

### 3. 顏色量尺 (`scale_color_manual`)

- **`values`**：強烈建議使用「具名向量」：`c("A" = "blue", "B" = "red")`。這能保證顏色精準對應組別，不因資料排序改變。

## 🖼️ 第四部分：主題 (Theme) 與細節修飾

`theme()` 負責所有「與數據無關」的視覺設定：

- **對齊控制 (`hjust`)**：0 = 靠左，0.5 = 置中，1 = 靠右。
    
- **文字設定 (`element_text`)**：
    
    - `size`: 大小；`face`: 粗細/斜體 ("bold", "italic")；`color`: 顏色。
        
    - `angle`: 旋轉角度 (常用於 X 軸標籤)。
        
    - **`margin`**：使用 `margin(t, r, b, l)` 調整與軸線的間距 (top, right, bottom, left)。
        
- **刻度控制 (`axis.ticks.length`)**：
    
    - 正數為向外，負數為向內（如 `unit(-0.1, "cm")`）。
        
- **長寬比 (`aspect.ratio`)**：
    
    - 固定畫布比例（高/寬），這對於確保存檔後的圖形不變形非常有用。

## 🚀 撰寫時的「防呆」清單

1. **加號位置**：`+` 必須留在**每一行的結尾**，不能在新行開頭。
    
2. **映射 vs 設定**：
    
    - 如果要根據組別變色，`color = Group` 要寫在 `aes()` 裡面。
        
    - 如果要全部統一顏色，`color = "blue"` 要寫在 `aes()` 外面。
        
3. **動態運算順序**：先用 `pretty()` 或 `range()` 算出漂亮的刻度與範圍，再把變數丟進 `scale_y_continuous`。
    
4. **存檔預覽**：使用 `ggsave` 時，務必指定 `width` 與 `height`，因為在 RStudio 視窗看到的樣子與輸出後的實體檔案可能因解析度不同而有差異。

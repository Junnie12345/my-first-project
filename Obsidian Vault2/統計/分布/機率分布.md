| 分佈類型                                | 常見應用情境                      | 機率分佈 (Exponential Family)                    | 常用連結函數 (Link Function)                                   | 典範連結 (Canonical Link) | 模型結構                                                | 參數意義與解釋                                             |
| ----------------------------------- | --------------------------- | -------------------------------------------- | -------------------------------------------------------- | --------------------- | --------------------------------------------------- | --------------------------------------------------- |
| **常態分佈 (Normal distribution)**      | 連續型資料，如體重、血壓、智商等            | ( Y \sim N(\mu, \sigma^2) )                  | 恒等連結 (Identity link): ( \mu = \eta )                     | ✅ 是                   | ( Y = \beta_0 + \beta_1 X + \epsilon )              | 每增加一單位 (X)，(Y) 平均變化 (\beta_1) 單位。                   |
| **二項分佈 (Binomial distribution)**    | 二元結果，如成功/失敗、生病/沒生病          | ( Y \sim Binomial(n, p) )                    | logit: ( \log\frac{p}{1-p} = \eta )；其他選項：probit、cloglog  | ✅ logit               | ( \log\frac{p}{1-p} = \beta_0 + \beta_1 X )         | (\exp(\beta_1)) 為勝算比 (odds ratio)；(\beta_1) 為對數勝算比。 |
| **卜瓦松分佈 (Poisson distribution)**    | 計數資料，如事件次數、住院次數、突變數量        | ( Y \sim Poisson(\lambda) )                  | log: ( \log(\lambda) = \eta )                            | ✅ log                 | ( \log(\lambda) = \beta_0 + \beta_1 X )             | (\exp(\beta_1)) 為率比 (rate ratio)，表示事件率變化倍數。         |
| **伽瑪分佈 (Gamma distribution)**       | 正值連續資料，常用於花費、時間、反應時間等（偏右分佈） | ( Y \sim Gamma(\mu, k) )                     | reciprocal: ( 1/\mu = \eta )；或 log: ( \log(\mu) = \eta ) | ✅ reciprocal          | ( g(\mu) = \beta_0 + \beta_1 X )                    | 可用於「反比關係」：X 增加使平均值下降。                               |
| **逆高斯分佈 (Inverse Gaussian)**        | 生存時間、加速度故障模型等（右偏正值資料）       | ( Y \sim IG(\mu, \lambda) )                  | reciprocal squared: ( 1/\mu^2 = \eta )                   | ✅ 是                   | ( g(\mu) = \beta_0 + \beta_1 X )                    | 適用於反應時間、存活時間模型。                                     |
| **多項分佈 (Multinomial distribution)** | 多類別結果，如診斷類型、選擇偏好            | ( Y \sim Multinomial(p_1, p_2, \dots, p_k) ) | generalized logit                                        | ✅ generalized logit   | ( \log\frac{p_j}{p_K} = \beta_{0j} + \beta_{1j} X ) | 每類別與參考類別之間的勝算比。                                     |
| **負二項分佈 (Negative Binomial)**       | 適用於「過度離散」計數資料（變異 > 平均）      | ( Y \sim NB(\mu, \theta) )                   | log: ( \log(\mu) = \eta )                                | ✅ log                 | 與 Poisson 相似，但允許額外變異                                | (\exp(\beta_1)) 為率比，但考慮額外散布參數。                      |
| **二項時間事件 (Bernoulli / Logistic)**   | 單次事件（n=1）之二元結果              | ( Y \sim Bernoulli(p) )                      | logit 或 probit                                           | ✅ logit               | 同 logistic regression                               | 勝算比解釋相同。                                            |

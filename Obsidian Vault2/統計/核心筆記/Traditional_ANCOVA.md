# 共變數分析 (ANCOVA)

> 🔗 返回：[[2. Statistical_Models_Map]]
> 📌 相關：[[ANOVA_Family]] | [[Regression_on_Change_Scores]]

---

## 一、定義

ANOVA + 連續共變數。目的為「控制共變數 (如基線值/年齡) 影響後，比較組別效果」。

- 是 **多元線性迴歸 (MLR)** 的特例
- 核心假設：殘差常態分佈、變異數同質性、**迴歸斜率同質性**

---

## 二、檢定類型與 ANCOVA

| **檢定類型**          | **變異歸屬原則**                                            | **適用情境 (程式碼順序)**                                      |
| ----------------- | ----------------------------------------------------- | ----------------------------------------------------- |
| **Type I (順序)**   | **第一個進入模型者優先瓜分變異。** 檢測一個變數時，**不控制**模型中**在它之後**進入的變數。  | 適用於有明確**層次或理論順序**的模型。R 的預設 `anova()` 函數就是 Type I。     |
| **Type II (階層)**  | 檢測一個變數時，**控制**模型中**所有不含該變數**的低階效應 (例如只控制主效應，不控制交互作用)。 | 適用於**無顯著交互作用**的主效應模型。                                 |
| **Type III (邊緣)** | 檢測一個變數時，**控制**模型中**所有其他項**，包括**交互作用項**。               | **最常用於 ANCOVA 和不平衡設計。** 檢測的是「在控制了模型中所有其他變數後，該項的獨特貢獻」。 |

> [!IMPORTANT]
> ANCOVA 應使用 **Type III SS**，確保共變量被正確調整。

---

## 三、R 語言實作

```r
library(car)

# ANCOVA 模型
model <- lm(DV_post ~ Group + DV_pre, data = df)

# Type III (SPSS 等價)
options(contrasts = c("contr.sum", "contr.poly"))
Anova(model, type = 3)
```

---

## 四、與改變量迴歸的關係

在數學上，ANCOVA（控制前測）與 [[Regression_on_Change_Scores]]（以差值為依變項並控制前測）是**等價的**。改變量迴歸可額外加入其他動態改變量（如 ΔMEQ）來探討機轉。

---

## 五、適用情境

- 前後測設計，控制基線差異
- 控制連續共變數（如年齡、BMI）的影響
